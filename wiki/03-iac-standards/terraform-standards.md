# Terraform IaC Standards

## Module Directory Layout

Every module in `infra-migration-patterns` must follow this structure:

```
modules/<category>/<module-name>/
├── main.tf          # Resource definitions only
├── variables.tf     # Input declarations with validation blocks
├── outputs.tf       # Output values
├── versions.tf      # required_providers + required_version constraints
└── README.md        # Auto-generated by terraform-docs on every merge
```

## Naming Convention

| Resource | Pattern | Example |
|----------|---------|---------|
| Virtual Network | `vnet-{workload}-{env}-{location_short}` | `vnet-hub-prod-uks` |
| Subnet | Azure-required or `snet-{purpose}-{env}` | `snet-app-prod` |
| Resource Group | `rg-{workload}-{env}` | `rg-workload-app-prod` |
| Storage Account | `st{workload}{env}{seq}` (no hyphens) | `stmigrationprod01` |
| Key Vault | `kv-{workload}-{env}-{location_short}` | `kv-app-prod-uks` |

## Tagging Requirements

All resources MUST include the following tags. Missing tags fail the Policy gate:

```hcl
locals {
  common_tags = {
    Environment   = var.environment    # dev | qa | prod
    WorkloadName  = var.workload_name  # e.g. "app-tier"
    CostCentre    = var.cost_centre    # e.g. "CC-MIGRATION-PROD"
    Owner         = var.owner          # team email
    MigrationWave = var.migration_wave # Wave-1 | Wave-2 | Wave-3 | Platform
    ManagedBy     = "Terraform"
  }
}
```

## Coding Rules

1. Never use `count` with modules — use `for_each` with a map
2. Never hardcode subscription IDs or tenant IDs — use data sources
3. Always add `validation` blocks to critical variables (environment, location)
4. Remote state only — no local `terraform.tfstate` files committed
5. Use `sensitive = true` on outputs that contain secrets
6. Run `terraform-docs` before raising a PR — README must be current

## Version Pinning

```hcl
# versions.tf — required in every module and root config
terraform {
  required_version = ">= 1.5.0"
  required_providers {
    azurerm = {
      source  = "hashicorp/azurerm"
      version = "~> 3.100"   # Minor updates allowed, major pin required
    }
    azuread = {
      source  = "hashicorp/azuread"
      version = "~> 2.47"
    }
  }
}
```
